# Katarsis Landing Page

Одностраничный лендинг для презентации цифрового продукта Katarsis — апсайкл-бренда для высокочувствительных людей.

## 🎯 Цель проекта

Создать интерактивный лонгрид, который:
- постепенно погружает пользователя в проблему
- вызывает эмоциональный отклик
- демонстрирует ценность продукта
- убеждает инвесторов

## 💡 Концепция

Лендинг построен как эмоциональное путешествие:
перегруз → осознание → катарсис → рациональное обоснование

## 🧩 Структура

- Hero (эмоциональный вход)
- Проблема
- Решение
- Продукт и фичи
- Пользователь
- User Flow
- Демо
- Бизнес-модель
- Рынок
- Финансы
- KPI
- CTA

## ⚙️ Технологии

- HTML5
- CSS3
- Vanilla JS

## 🎮 Интерактив

- Плавный скролл
- Демонстрация эмоционального сценария
- Карточки и таблицы

## 📊 Бизнес-часть

Включает:
- модель монетизации
- финансовую таблицу
- оценку окупаемости
- KPI
